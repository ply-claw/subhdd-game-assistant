# SubHDD Game Assistant

Chrome 扩展，为 [sub.hdd.sb](https://sub.hdd.sb/) 游戏中心的五款益智游戏提供辅助功能。

## 支持的游戏

| 游戏 | 难度 | 状态 |
|------|------|------|
| **🧩 2048** | 3×3 (入门) | ⚠️ 算法可运行，胜率待提升 |
| **🧩 2048** | 4×4 (经典) | ✅ 正常 |
| **🧩 2048** | 5×5 (挑战) | ✅ 可达到目标 |
| **🃏 记忆翻牌** | 全部 (3×4 / 4×4 / 4×6 / 6×6) | ✅ 正常 |
| **🧮 华容道** | 全部 (3×3 / 4×4 / 5×5) | ✅ 正常 |
| **🔢 数独** | 全部 (入门 / 经典 / 挑战 / 专家) | ✅ 正常 |
| **🐑 羊了个羊** | 全部 (简单 / 普通 / 困难 / 地狱) | ✅ 正常 |

### 面板功能

- **🧩 2048** — 显示推荐滑动方向（Expectimax 搜索，深度可调 1-5），自动完成
- **🃏 记忆翻牌** — 追踪已翻卡面并显示配对网格，自动完成
- **🧮 华容道** — 显示解法步骤列表（IDA\* / 三阶段 A\*），自动完成
- **🔢 数独** — 显示完整 9×9 答案网格，自动填入
- **🐑 羊了个羊** — DFS 回溯一次性求解完整消除序列，自动完成

## 安装

1. 下载代码或 clone 本仓库
2. 打开 Chrome，地址栏输入 `chrome://extensions`
3. 右上角开启「**开发者模式**」
4. 点击「**加载已解压的扩展程序**」
5. 选择项目根目录（包含 `manifest.json` 的文件夹）

## 使用

### 前置条件

先在 Chrome 中登录 [sub.hdd.sb](https://sub.hdd.sb/)。扩展会自动读取登录状态，无需额外配置。

### 单局辅助

打开任意游戏页面（如 2048），页面右侧会出现辅助面板：

- **显示解法** — 根据当前棋盘给出最优提示
- **自动完成** — 自动操作完成当前这局
- **停止** — 随时中断自动操作

### 一键每日全通

点击 Chrome 工具栏的扩展图标，在弹窗中：

1. 查看当日各游戏剩余次数和签到状态
2. 选择运行模式（新标签页 / 当前标签）
3. 调整 2048 搜索深度（默认 3）
4. 点击「**🚀 一键全通**」

**新标签页模式**（默认）：后台自动打开独立标签页，签到→记忆翻牌→数独→华容道→羊了个羊→2048，每游戏完成全部难度后关闭。

**本标签模式**：在当前标签页内依次导航完成。

#### 定时自动通关

开启「每日定时自动通关」并设置时间（默认 01:00），扩展定时自动执行全通，适合长期挂机。

## 原理

扩展通过 DOM 驱动读取游戏状态、派发键盘/点击事件操作游戏。所有操作经过页面自身的网络请求，不额外访问后端。自动操作间加入随机延迟模拟真人。华容道采用 IDA\* + 递归降维回退求解，羊了个羊用 DFS 回溯遍历覆盖图，2048 用 Expectimax 搜索（移植自 2048EndgameTablebase C++ 算法）。

## 开发

纯 Vanilla JS，Manifest V3，无构建步骤，无外部依赖。

```
├── manifest.json
├── content/
│   ├── main.js              # Content script 入口
│   ├── inject.js            # 注入主世界的桥接脚本
│   ├── runner.js            # 每日一键全通协调器
│   ├── ui/
│   │   ├── panel.js         # 侧边栏 UI 逻辑
│   │   └── panel.css        # 侧边栏样式
│   └── solvers/
│       ├── solver2048.js    # Expectimax 求解器
│       ├── solverMemory.js  # 配对追踪
│       ├── solverPuzzle15.js # A* 求解器
│       └── solverSudoku.js  # AC-3 + Backtracking
├── popup/
│   ├── popup.html
│   ├── popup.js
│   └── popup.css
└── icons/
```
