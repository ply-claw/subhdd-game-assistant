# SubHDD Game Assistant

Chrome 扩展，为 [sub.hdd.sb](https://sub.hdd.sb/) 游戏中心的四款益智游戏提供辅助功能。

## 支持的游戏

| 游戏 | 难度 | 状态 |
|------|------|------|
| **🧩 2048** | 3×3 (入门) | ⚠️ 算法可运行，胜率待提升 |
| **🧩 2048** | 4×4 (经典) | ✅ 正常 |
| **🧩 2048** | 5×5 (挑战) | ✅ 可达到目标 |
| **🃏 记忆翻牌** | 全部 (3×4 / 4×4 / 4×6 / 6×6) | ✅ 正常 |
| **🧮 华容道** | 3×3 (入门) | ✅ 正常 |
| **🧮 华容道** | 4×4 (经典) | ✅ 正常 |
| **🧮 华容道** | 5×5 (挑战) | ✅ 正常 |
| **🔢 数独** | 全部 (入门 / 经典 / 挑战 / 专家) | ✅ 正常 |

### 面板功能

- **🧩 2048** — 显示推荐滑动方向（Expectimax 搜索，深度可调 1-5），自动完成
- **🃏 记忆翻牌** — 追踪已翻卡面并显示配对网格，自动完成
- **🧮 华容道** — 显示解法步骤列表（IDA\* / 三阶段 A\*），自动完成
- **🔢 数独** — 显示完整 9×9 答案网格，自动填入

## 安装

1. 下载代码或 clone 本仓库
2. 打开 Chrome，地址栏输入 `chrome://extensions`
3. 右上角开启「**开发者模式**」
4. 点击「**加载已解压的扩展程序**」
5. 选择项目根目录（包含 `manifest.json` 的文件夹）

## 使用

### 前置条件

先在 Chrome 中登录 [sub.hdd.sb](https://sub.hdd.sb/)。扩展会自动读取登录状态，无需额外配置。

### 单局辅助

打开任意游戏页面（如 2048），页面右侧会出现辅助面板：

- **显示解法** — 根据当前棋盘给出最优提示
- **自动完成** — 自动操作完成当前这局
- **停止** — 随时中断自动操作

### 一键每日全通

点击 Chrome 工具栏的扩展图标，在弹窗中：

1. 查看当日各游戏剩余次数
2. 调整 2048 搜索深度（默认 3）
3. 点击「**🚀 一键全通**」

扩展会自动完成：签到 → 2048（全部难度）→ 记忆翻牌 → 华容道 → 数独。请保持页面打开，预计 3-8 分钟。

## 原理

扩展通过注入脚本到页面主世界，直接读取游戏状态并调用页面自身的 API 函数。所有操作经过页面自身的网络请求，不额外访问后端。自动操作间加入 200-800ms 随机延迟，模拟真人。

## 开发

纯 Vanilla JS，Manifest V3，无构建步骤，无外部依赖。

```
├── manifest.json
├── content/
│   ├── main.js              # Content script 入口
│   ├── inject.js            # 注入主世界的桥接脚本
│   ├── runner.js            # 每日一键全通协调器
│   ├── ui/
│   │   ├── panel.js         # 侧边栏 UI 逻辑
│   │   └── panel.css        # 侧边栏样式
│   └── solvers/
│       ├── solver2048.js    # Expectimax 求解器
│       ├── solverMemory.js  # 配对追踪
│       ├── solverPuzzle15.js # A* 求解器
│       └── solverSudoku.js  # AC-3 + Backtracking
├── popup/
│   ├── popup.html
│   ├── popup.js
│   └── popup.css
└── icons/
```
